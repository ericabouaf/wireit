Constructor

Properties

layer - object

options - object

Options:
  • parentEl: parent element (defaut layer.el)
  • className: default to "WireIt-LayerMap"
  • style: display style, default to "rgba(0, 0, 200, 0.5)"
  • lineWidth: default 2
